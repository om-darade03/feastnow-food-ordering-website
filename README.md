# 🍽️ FeastNow – Online Food Ordering Website

A fully responsive online food ordering website built with HTML, CSS, Bootstrap, and JavaScript.

## 📋 Project Description

FeastNow is an attractive, feature-rich online food ordering website where users can:
- Browse restaurant menus organized by category
- Add items to cart with quantity controls
- View a live order summary with pricing
- Place orders with a success confirmation modal

## 🛠️ Technologies Used

| Technology  | Purpose                          |
|-------------|----------------------------------|
| HTML5       | Page structure & semantic markup |
| CSS3        | Custom styles, animations, layout |
| Bootstrap 5 | Responsive grid, components      |
| JavaScript  | Cart logic, filtering, interactivity |
| Font Awesome | Icons                            |
| Google Fonts | Typography (Playfair Display + DM Sans) |

## 📁 Project Structure


## ✨ Features

- **Hero Section** – Animated landing with floating badges and stats
- **Category Filter** – Filter by All, Pizza, Burgers, Sushi, Pasta, Desserts, Drinks
- **Menu Grid** – 18 food items with emoji visuals, ratings, descriptions
- **Shopping Cart** – Offcanvas sidebar with add/remove/quantity controls
- **Order Summary** – Real-time subtotal and total calculation
- **Order Success Modal** – Order tracking steps after placing order
- **Contact Form** – With toast notification feedback
- **Responsive Design** – Works on mobile, tablet, and desktop
- **Smooth Animations** – Scroll animations, hover effects, floating badges

## 🚀 How to Run

1. Unzip the project folder
2. Open `index.html` in any modern web browser
3. No server or installation required!

## 📱 Responsive Breakpoints

- Mobile: < 576px
- Tablet: 576px – 992px
- Desktop: > 992px
