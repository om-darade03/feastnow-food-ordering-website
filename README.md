# 🍽️ FeastNow – Online Food Ordering Website

A fully responsive online food ordering website built with *HTML, CSS, Bootstrap 5, and JavaScript*.

## 🚀 Live Demo
> Open index.html in any browser – no installation required!

## 📋 Description
FeastNow is an attractive food ordering website where users can browse restaurant menus, add items to the cart, and view order summaries with a modern animated UI.

## ✨ Features
- 🏠 Hero section with animated floating badges
- 🗂️ Category filter – Pizza, Burgers, Sushi, Pasta, Desserts, Drinks
- 🍽️ Menu grid with 18 food items, ratings & descriptions
- 🛒 Shopping cart with add, remove & quantity controls
- 💰 Real-time order summary with total calculation
- 🎉 Order success modal with tracking steps
- 📬 Contact form with toast notification
- 📱 Fully responsive – Mobile, Tablet & Desktop

## 🛠️ Technologies Used
| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling & animations |
| Bootstrap 5 | Responsive layout |
| JavaScript | Cart logic & interactivity |
| Font Awesome | Icons |
